**Check Current Assignments:**

**Complete Tasks:**
1. Receive assignment via @symbol
2. Execute task according to agent specialization
3. Mark complete: Change `[ ]` to `[x]` 
4. Add completion notes if needed

**Example Completion:**
